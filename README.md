# Pandoc jupyter

## Requirements

- Pandoc
- TeX Live
- Jupyter
- [watchexec](https://github.com/watchexec/watchexec) - For live reload

## Requirements for this particular homework

- Numpy
- Matplotlib

## Compilation

```shell
$ make -j3 build/output.pdf
```

## Live reload

```shell
$ make live
```

## Clean artifacts

```shell
$ make clean
```
